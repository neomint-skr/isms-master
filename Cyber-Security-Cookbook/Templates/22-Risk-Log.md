# Risk Log

This template defines the Risk Log structure (LRM-44, 4×4). The Risk Log is the SSOT for risks and includes the treatment view.

## Columns (minimum)

| Column | Value |
|---|---|
| RISK-ID | [RISK-001] |
| Scope Unit | [SU-### — Name] |
| Risk scenario | [One sentence] |
| CIA focus | [C/I/A] |
| Impact | [Minor(1)/Moderate(2)/Major(3)/Severe(4)] |
| Impact notes | [One line] |
| Likelihood | [Rare(1)/Unlikely(2)/Possible(3)/Likely(4)] |
| Likelihood notes | [One line] |
| Score | [I×L] |
| Band | [Low/Medium/High/Critical] |
| Decision | [Accept/Treat/Transfer/Avoid] |
| Risk owner | [Role/Name] |
| Controls selected | [Annex A refs and/or SM-IDs] |
| Actions | [Max 1–3 ticket links if treated] |
| Target residual | [e.g. ≤ Medium (≤6)] |
| Due date | [YYYY-MM-DD] |
| Approval | [Name/Role + date] |
| Last review | [YYYY-MM-DD] |
